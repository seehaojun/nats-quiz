# Nat's Quiz

Interactive P4 Science quiz app covering the Diversity topic for Primary 4 students.

## Features

- Multiple quiz topics: Classifying Living Things, Fungi & Bacteria, Flowering vs Non-Flowering Plants, Life Cycles, and more
- Instant feedback with explanations for each answer
- Score tracking with progress bar
- Review mode to see correct answers after completing a quiz
- Play individual topics or all questions combined

## Tech

Single-file HTML app — no build tools or dependencies. Just open `index.html` in a browser.

## Usage

```
open index.html
```

Or visit via GitHub Pages if enabled.
